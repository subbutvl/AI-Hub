## 🌟 Overview

**AI Hub** is a premium developer tool designed to streamline the lifecycle of AI projects. Whether you are searching for the latest LLM repositories, curating educational YouTube content, or building complex AI skill pipelines, AI Hub provides a unified, high-performance interface to manage it all.

---

## 🛠️ Key Features

### 📊 Mission Control Dashboard
- **Real-time Analytics**: Instant visibility into your total repositories, web links, and active AI skills.
- **Dynamic Summaries**: Adaptive mission statements that keep you focused on your project goals.
- **Recent Activity**: Quick-access widgets for your latest additions and high-impact use cases.

### 📂 Repo Hub (The Ultimate Index)
- **Deep Integration**: Seamlessly add and explore GitHub repositories.
- **Categorization & Tagging**: Organize your research with custom folders and multi-tag filtering.
- **File Explorer**: Browse repository structures directly within the app to understand project architectures instantly.

### 🌐 Web Hub (Smart Curation)
- **Intelligent Detection**: Auto-detects YouTube videos, channels, playlists, and GitHub links.
- **Metadata Engine**: Automatically fetches titles, descriptions, and thumbnails to keep your library rich and searchable.
- **Dual View Layout**: Switch between a visual **Card Grid** for discovery and a dense **Table List** for rapid management.
- **Import/Export**: Full portability via CSV for your entire link collection.

### ⚡ Skill Hub & Use Cases
- **Skill Creator**: Build custom prompted skills with versioning and language support.
- **Pipeline Builder**: Chain multiple skills together into powerful automated workflows.
- **Ready-to-Use Templates**: Pre-configured use cases for Code Reviews, Security Analysis, and Content Summarization.

## 🚀 Getting Started

### Prerequisites
- **Node.js** (v18.0.0 or higher)
- **npm** or **yarn**

### Local Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/ai-hub.git
   cd ai-hub
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Configure Environment:**
   Update the `.env` file with your API keys (e.g., Gemini API Key).
   ```bash
   GEMINI_API_KEY=your_key_here
   ```

4. **Start Development Server:**
   ```bash
   npm run dev
   ```
   Open [http://localhost:5173](http://localhost:5173) to view the app.
